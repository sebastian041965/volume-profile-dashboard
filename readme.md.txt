# 📊 Volume Profile Dashboard

Bienvenido a tu espacio de análisis técnico profesional. Esta app te permite visualizar el perfil de volumen, trazado técnico, dibujo libre y gráficos de velas con indicadores personalizados, todo en una interfaz segura y dinámica.

---

## ✨ Funcionalidades

- 🔐 Autenticación por usuario con sesión limitada
- 📊 Perfil de volumen con VPOC, soporte y resistencia
- 📈 Gráfico de velas con temporalidades desde 1 minuto hasta mensual
- 📐 Media móvil configurable (SMA, EMA, WMA) con desplazamiento
- 🖌️ Dibujo libre editable con exportación/importación
- 🔄 Detección automática de fuente: Binance para criptos, Yahoo Finance para divisas

---

## 🚀 Cómo desplegar en Streamlit Cloud

1. Sube este repositorio a tu cuenta de GitHub.
2. Ve a [streamlit.io/cloud](https://streamlit.io/cloud) e inicia sesión.
3. Haz clic en “New app”.
4. Selecciona este repositorio.
5. Usa `volume_profile_app.py` como archivo principal.
6. ¡Tu app estará online!
